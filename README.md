# 🔐 Advanced Network Security Scanner

A Python-based GUI network security scanner with powerful features like:

## 🚀 Features
- 🔍 ARP-based host discovery
- ⚡ Multithreaded port scanning
- 🧠 Banner grabbing
- 🛡️ Basic vulnerability detection
- 🌙 Dark theme GUI (Tkinter)

## 🛠️ Technologies Used
- Python
- Scapy
- Tkinter
- Socket Programming

## 📦 Installation

```bash
git clone https://github.com/jeshu007/Advanced-Network-Scanner.git
cd Advanced-Network-Scanner
pip install -r requirements.txt
python gui.py
